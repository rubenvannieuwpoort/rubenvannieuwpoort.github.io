Gaussian quadrature and orthogonal polynomials

Suppose we want to compute the integral

∫f(x) μ(x) dx

many different times for different functions f. Here, μ(x) is a nonnegative weight function.

For some f we might be able to evaluate the integral analytically (i.e. by integrating and obtaining an exact answer). However, in computational contexts it is often both more efficient and simpler to use quadrature rule.

Definition: A quadrature rule is a collection of weights w1​,w2​,...,wn​ and arguments x1​,x2​,...,xn​ such that

k=0∑n−1​wk​f(xk​)≈∫f(x) μ(x) dx

If, for some f, the approximation is exact (i.e. the approximation symbol in the definition can be replaced by an equality symbol), we say that the quadrature rule is exact for f.

In general, a quadrature rule can give a good approximation of the integral for some f and a bad one for other f. In practice, we often require the quadrature rule to be exact for some set of functions, often polynomials up to some maximum degree. We call this the exactness conditions. By the linearity of the integral, if a quadrature rule is exact for functions f1​,f2​,..., the quadrature rule is also exact for any linear combination of f1​,f2​,....

In the following, we will denote the space of polynomials with degree up to n as Pn​.

Quadrature rules, the easy way

It is not too complicated to derive quadrature rules that are exact for any polynomial p∈Pn​ for a given n. Given a function f and d+1 distinct points x0​,x1​,...,xd​, we can use Lagrange interpolation to construct a polynomial fd​ of degree d that interpolates f at x0​,x1​,...,xd​. That is,

fd​(xk​)=f(xk​) for k=0,1,...,d

The polynomial fd​ is defined as fd​(x)=∑k=0n​f(xk​)lk​(x), where lk​ are the Lagrange polynomials

lk​(x)=(xk​−x0​)(xk​−x1​)⋯(xk​−xk−1​)(xk​−xk+1​)⋯(xk​−xd​)(x−x0​)(x−x1​)⋯(x−xk−1​)(x−xk+1​)⋯(x−xd​)​

We can now integrate the interpolation polynomial fd​ of f as

∫p(xk​) μ(x) dx=∫p(xk​) μ(x) dx=∫k=0∑n​f(xk​)lk​(x) μ(x) dx

Bringing the summation and the terms f(xk​) out of the integral, we see that

∫fd​(xk​) μ(x) dx=k=0∑n​f(xk​)∫lk​(x) μ(x) dx

So, if we set wk​=∫lk​(x) μ(x) dx, we have

∫fd​(xk​) μ(x) dx=k=0∑n​wk​f(xk​)

Now, since fd​ interpolates f, we expect that fd​≈f, so that

∫fd​(x) μ(x) dx≈∫f(x) μ(x) dx=k=0∑n​wk​f(xk​)

So we have found a quadrature rule that is exact for any f∈Pd​.

It is also possible to find the weights in a more computational way. If we want the quadrature rule to be exact for function space of dimension d, we can pick a basis of d functions, pick d distinct positions x1​,x2​,...,xd​, write out the exactness requirements, and interpret the resulting system of equations with the weights as the unknowns.

For example, suppose that we want the quadrature rule to be exact for any p∈Pd​. We pick d+1 basis functions b0​,b1​,...,bd​ (for example, bk​(x)=xk would work) and distinct positions x0​,x1​,...,xd​. Then, the exactness requirements become

j=0∑d​wj​bk​(xj​)=∫bk​(x) μ(x) dx for k=0,1,...,d

which is a linear system of d+1 equations and d+1 unknowns w0​,w1​,...,wd​. In particular, if we use bk​(x)=xk the resulting system is a Vandermonde system.

For now, we have regarded the positions x1​,...,xd​ as given. If we change our point of view and interpret them as unknowns we can interpret the exactness requirements as a nonlinear system in 2d unknowns. If we expect this system to behave like a non-signular linear one (there is no good reason to expect this, but let’s be optimistic), we would expect to be able to solve a system of 2d equations with it. This is by analogy with a linear system, where a non-singular system has a unique solution if there are as many unknowns as equations.

So, this would mean that quadrature rules with n points exists that are exact for any p∈P2n−1​. These quadrature uses roughly half the number of points of the quadrature rules derived via the method presented earlier, at the cost of not having the freedom of choosing the positions x0​,x1​,... freely.

In fact, it is possible to show that such quadrature rules exist, using orthogonal polynomials.

Orthogonal polynomials

Consider the following inner product on a space of functions:

⟨f,g⟩=∫f(x)g(x) μ(x) dx

Like before, μ is a nonnegative weight function. This forms a vector space, and the usual notion of orthogonality applies.

Definition: A sequence of polynomials p0​,p1​,... is orthogonal if

  1. deg(pk​)=k
  2. ⟨pi​,pj​⟩=0 whenever i=j

Like with vectors, we call the sequence orthonormal if additionally ⟨pk​,pk​⟩=1.

Lemma 1: If p is a polynomial with deg(p)<n, and p0​,p1​,... is a sequence of orthogonal polynomials, then

⟨p,pn​⟩=0

Proof: The polynomials p0​,p1​,...,pn−1​ form a basis of Pn​. Since p∈Pn​, we can write p=∑k=0n−1​ck​pk​. It follows that ⟨p,pn​⟩=⟨∑k=0n−1​ck​pk​,p⟩=∑k=0n−1​ck​⟨pk​,pn​⟩. Since p0​,p1​,... is an orthogonal sequence of polynomials and k<n, all terms in the summation are zero, so it follows that ⟨p,pn​⟩=0.
□

There is the following nice result about the zeroes of orthogonal polynomials.

Lemma 2: Let p0​,p1​,... be a sequence of orthogonal polynomials with respect to the inner product ⟨f,g⟩=∫f(x)g(x) μ(x) dx, and let the support of μ be contained in [a,b]. Then pn​ has n distinct roots x1​,x2​,...,xn​∈(a,b).

Proof: Let x1​,x2​,...,xm​ be the zeroes of pn​ with odd multiplicity and lying in (a,b). Define q(x)=(x−x1​)(x−x2​)⋯(x−xm​). Now q(x)pn​(x) does not change sign on [a,b], so ⟨q,pn​⟩=0. By lemma 1, it follows that deg(q)=n. So m=n, which implies that pn​ has n distinct roots in (a,b).
□

Now we are ready to prove the following theorem.

Theorem: Let p0​,p1​,... be a sequence of orthogonal polynomials with respect to the inner product ⟨f,g⟩=∫f(x)g(x) μ(x) dx, and let x0​,x1​,...,xn​ be the zeroes of pn​. Define wk​=∫lk​(x) μ(x) dx for k=0,1,...,n with l0​,l1​,...,ln​ the Lagrange polynomials for x0​,x1​,...,xn​. Then

k=0∑n​wk​f(xk​)=∫f(x) μ(x) dx

for f∈P2n−1​

Proof: Let fn​∈Pn​ be the polynomial of degree n that interpolates f at x0​,x1​,...,xn​. It was shown earlier that

∫fn​(x) μ(x) dx=k=0∑n​wk​f(xk​)

Now consider the integral

∫(f(x)−fn​(x)) μ(x) dx

We want to show that this integral is zero, since this is equivalent to ∫f(x) μ(x) dx=∫fn​(x) μ(x) dx. We have seen

First, we observe that f(x)−fn​(x) has zeroes at x0​,x1​,...,xn​. So f(x)−fn​(x) has the same zeroes as pn​ and must be some multiple of pn​, and we can write f(x)−fn​(x)=cpn​(x)q(x) for some q∈Pn−1​.

Now, we can write the integral as ∫(f(x)−fn​(x)) μ(x) dx=c∫pn​(x)q(x) μ(x) dx=⟨pn​,q⟩. Since deg(q)<n we have ⟨pn​,q⟩=0 by lemma 1.
So it follows that

∫f(x) μ(x) dx=k=0∑​wk​f(xk​)

Because both are equal to ∫fn​(x) μ(x) dx.
□
