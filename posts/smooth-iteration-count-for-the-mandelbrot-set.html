Smooth iteration count for the Mandelbrot set

The exterior of the Mandelbrot set is usually colored by using the number of iterations. As the number of iterations is a discrete quantity, this leads to the occurrence of banding in the coloring. In this article, I will derive a formula for a fractional iteration count. This is by no means novel; plenty of derivations can be found on the internet. However, I found most derivations to be fairly hard to follow, and hope that my explanation may be helpful to at least some people.

It is not hard to see that the smooth iteration count is continuous. Imagine we are picking a starting point in a fractal colored with the discrete iteration count. Suppose the starting point has iteration count nnn. If we move the starting point z0z_0z0​ around a little bit, but no so much that the discrete iteration count changes, the smooth iteration count is a continuous function of z0z_0z0​. We can visualize this as moving the starting point without leaving the 'band' that the starting point is in. When we move the starting point so much that it approaches the outer edge of its band, we see that the smooth iteration count will approach nnn. Likewise, if we move the starting point near the inner edge of the of the band, the smooth iteration count will approach n+1n + 1n+1. This means that the smooth iteration count is continous, even as the starting point crosses bands, since the smooth iteration count will assume the value nnn on the border between the nnnth and the n+1n + 1n+1th band.
